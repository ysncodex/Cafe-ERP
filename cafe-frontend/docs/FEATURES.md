# Features

## Dashboard
- KPIs (sales/costs/funds)
- Charts and trends

## Transactions
- Daily Record: browse/search/filter records
- Daily Expense: track operational expenses

## Costs
- Product Costs (variable costs)
- Fixed Costs (recurring costs)

## Fund Management
- Track cash/bank/bkash movements

## Reports
- Summary KPIs
- Exports (PDF/Excel/CSV) via shared utilities

## Global UI systems
- Reusable modals for manager password + editing transactions
- Pagination hook + component
- Toast notifications
