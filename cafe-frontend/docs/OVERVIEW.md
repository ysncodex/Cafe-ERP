# Project Overview

This repository contains the frontend for a cafe ERP-style dashboard (transactions, costs, funds, and reporting).

## What this app does
- Login screen and protected dashboard area
- Dashboard KPIs and charts
- Daily Record browsing (filter/search + pagination)
- Daily Expense tracking
- Product (variable) cost tracking
- Fixed cost tracking
- Fund Management (add/withdraw/transfer/invest)
- Reports view with export utilities

## Key UX features
- Consistent, reusable UI components (buttons, cards, tables, modals, toasts)
- Global pagination utilities
- User-managed lists for categories/items/suppliers (persisted)
- Demo mode for guest usage (data swaps instantly)

## Tech stack
- React + TypeScript + Vite
- Tailwind CSS
- React Router
- react-hook-form + zod validation
- Recharts for charts
- Axios for API client utilities
