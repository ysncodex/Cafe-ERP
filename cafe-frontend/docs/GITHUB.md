# GitHub Notes

## Recommended repo docs
- `README.md`: short, polished landing page (what it is + how to run)
- `docs/OVERVIEW.md`: full product summary
- `docs/SETUP.md`: setup and environment
- `docs/ARCHITECTURE.md`: structure and how to extend

## Suggested screenshots
Recommended: add screenshots under `docs/assets/` and link them from `README.md`.

## Release hygiene (optional)
- Add `CHANGELOG.md` if you plan to version releases.
- Add `LICENSE` if you plan to open-source.
